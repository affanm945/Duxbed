# Duxbed Furniture Website

A full-stack furniture e-commerce style website developed using **React, PHP, and MySQL**.
The application allows users to browse furniture products and interact with a backend API for data management.

## Tech Stack

**Frontend**

* React
* JavaScript / TypeScript
* HTML5
* CSS3

**Backend**

* PHP

**Database**

* MySQL

## Features

* Furniture product listing
* Product details display
* Backend API integration using PHP
* MySQL database for storing product data
* Responsive UI built with React

## Project Structure

```
Duxbed

├── backend          # PHP backend APIs
├── public           # Static files

```

## Installation

### 1. Clone the Repository

```
git clone https://github.com/affanm945/Duxbed.git
```

### 2. Install Frontend Dependencies

```
npm install
```

### 3. Run React Development Server

```
npm run dev
```

### 4. Setup Backend

* Place the PHP backend inside your local server (XAMPP / WAMP / LAMP)
* Configure database connection in the PHP config file

### 5. Setup Database

Create a MySQL database and import the required tables for the furniture products.
